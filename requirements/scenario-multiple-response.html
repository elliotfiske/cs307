2.4.3. Creating a Multiple Response Question

To create a multiple response question, the instructor opens the question creator dialog, and then changes the question type to Multiple Response. The instructor will see Figure 6.

Figure 6: Default multiple response view.


Nearly all of the features from the multiple choice question type translate to the multiple response question type. The difference is that the instructor has the option of choosing multiple answers to be considered correct. To mark an answer as correct, the instructor selects the box next to the answer. If the instructor wants to change a marked box back to being incorrect, the instructor selects the box again.
