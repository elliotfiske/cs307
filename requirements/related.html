1.5. Related Systems

Some online test making/taking tools that provide functionality similar to the Test Tool include:

  • Blackboard
  • ExamProfessor
  • Canvas
  • ClassMarker
  • TestMoz
  • QuizGen

These systems offer great diversity in functionality regarding the creation and distribution of tests. The Test Tool aims to include the significant core features found in these tools, not the full range of features found in all of them. The Test Tool provides practical improvements in several areas that are not fully covered by most other available tools. These improvements are:

  • the ability to create questions that support programming code;
  • more customizability of how individual questions are organized;

A detailed review of each related tool follows in these subsections:
