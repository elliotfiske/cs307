1.5.6 quizgen

An open source test generation tool, quizgen, allows instructors to create assessments and then make them availble to students in html format for self-assessment purposes. Since the output is in html, this tool is ideal for generating tests that are compatible with all platforms. Unfortunately, due to the web-based output, the tool lacks functionality essential to our customer's needs. The student opens the html file and clicks on the answers to the questions, to which the document displays whether the student answered correctly.

Good Features:

  • its web-based tests are compatible with any device that has an internet connection
  • supports all the basic question types and has syntax-highlighting for code
  • supports special formatting and LateX
  • supports embedding images/media into the tests

Bad Features:

  • this tool only generates self-asessements, which is not ideal for exams
  • generating the tests is very complex and requires editing a text file
  • the generated tests are static and cannot be randomized to prevent cheating

Missing Features:

  • no test bank (very critical, a test generation tool is practically useless if the instructor has to create each test by hand)
  • does not report scores back to the instructor
  • no GUI for fast test generation
  • cannot adjust point values
  • does not allow students to have multiple tries
  • does not prevent unauthorized users from accessing the test
  • no time constraints
