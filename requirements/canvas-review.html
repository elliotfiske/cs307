1.5.3 Canvas LMS

Canvas is a Learning Management System (LMS) used in many large universities and K-12 institutions to make managing course materials easier on students and teachers. Assignment submissions and grading are also done through Canvas, so an entire class can be taught and exams administered through the LMS. While there are many features in Canvas that make teaching easier, the requirements for Test Tool described in these requirements does not require learning solutions.

Canvas looks to be a leader in the field of LMSs with their multi-platform application and advanced features.

Good Features:

  • Multiplatform (iOS, Android, Web, Desktop Application)
  • Easy Calendar Scheduling
  • Push notifications for students
  • Support for external education links (e.g. importing a Khan Academy Video)
  • Instructor audio/video comments
  • Scaleable, fast, low-risk cloud services
  • Analytics
  • File downloading and uploading
  • Customizable interface
  • Speedgrader with annotations
  • API

Bad Features:

  • Relies on cloud services
  • Requires internet connection
  • Limited assignment/testing features

Missing Features:

  • Proctoring support
  • Test timers
  • Anti-cheat methods
