1.5.1. ExamProfessor

ExamProfessor is a test-making and test-taking tool released, but partially still in development.

It is acceptable only for small quizzes. It supports multiple platforms by being in-browser.

Good Features:

  • Easy to use. Has drag-and-drop features. Question and test creation is fairly straightforward.
  • Easy to look at. Relatively uncluttered
  • CSV and Excel imports/exports, PDF exports
  • Strong level of management. Teachers can create classes and view student accounts

Bad Features:

  • Purely multiple choice. Very few options for question variety
  • Still in development, not widely used
  • Free version is incredibly limited

Missing Features:

  • Multiple question formats
  • Question categories
  • Question point values
  • No accomodation for code questions
