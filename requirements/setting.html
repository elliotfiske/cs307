1.3. Operational Setting

Test Tool will have 2 operational settings: (1) The software system being used by the teacher or proctor who creates and gives exams; (2) The software system being used by the student to take an exam.

Test Tool will be designed for private use within a classroom setting at a college or university. The software should be installed on all computers in the participating classes and students and teachers will both need accounts to access the software.

Depending on the class Test Tool is to be used in and the operating system is it to be installed on, different set up instructions may need to be followed.
