1.5.1. Blackboard

Blackboard is a comprehensive suite of webapps and software that helps teachers organize their classes. There are tools to create classes, manage students, assign grades, and do online tests. Cal Poly used Blackboard before switching to the new PolyLearn (Moodle) system. We will examine the major features of Blackboard below:

Blackboard is a large tool that encompasses some of the desired features in Test Tool, but does not have all of them.


Good Features of Blackboard
  • Platform for creating courses, signing up students, managing homework
  • Different software for College, K-12, Enterprise and Business situations - we'll focus on the features of the College suite
  • Forums for students to discuss homework and coordinate studying
  • Online test-creation, grading, cheat-prevention
  • Test Creation:
    • Link to Test Creation UI
    • You can create a "question bank" and create tests from these questions
    • Different test question formats, like multiple choice, fill-in, and definition matching
    • You can randomize question order and aspects like the order that multiple choices appear in
    • You can import test questions from textbook publishers
    • You can assign weighted points to a question
Missing Features of Blackboard
  • You can print tests, but the tests are formatted like a webpage and doesn't look very good
  • You can set a specific time period students can take the test, but it's not really a system for proctoring in-class exams
  • There is no option to "practice" an exam for students that aren't in a class
Administrative Details
  • Can either be self-hosted or hosted on the cloud (Saas)
  • Runs in-browser
  • Provides support for mobile devices
