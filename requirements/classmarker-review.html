1.5.4 ClassMarker Review

ClassMarker is a highly developed tool for giving tests. ClassMarker has question banks, easy test distribution, and automatic grading. The web-based snd therefore incredibly versatile as it can work across any device and in a variety of environments. However it is lacking one of the most crucial features of test tool - code based questions.

Overall ClassMarker actually has a fantastic set of features for test takers, but it's general use design makes it lack the necessary features for a code based exam.

Good Features

  • Test Banks
  • Auto Grading
  • Ability to distribute test publicly and privately
  • A variety of test formats
  • Question randomizations
  • Fairly nice GUI based question creation

Bad Features

  • Question importing is only specific to their format
  • Test randomization doesnâ€™t generate a test, rather gives every student a different test with the same types of questions with each test being different
  • Seems to be made with businesses in mind and as such has many unnecessary features

Missing Features

  • Test generation (as opposed to just randomization)
  • Support for coding based questions
  • Sharing questions between departments
  • Any sort of difficulty ratings
