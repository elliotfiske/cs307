1.5.5 TestMoz

TestMoz is a relatively popular test-making tool. It provides the absolute basics for test creation and prides itself on its simplicity. Users take the test through their Internet browser without any registration, and both the user and administrator can see the results as soon as the user is finished. However, its simplicity is also a flaw: customizability is low, and there is no true user identification.

TestMoz lacks many features that provide depth to the test creation process; yet, if the test in question does not need such complexity, this tool functions extremely well.

Good Features:

  • simple to operate
  • easy to distribute to users
  • no registration is needed
  • graded immediately
  • users can see the correct answers after the test, if the administrator allows it
  • provides a detailed overview of class performance
  • can export results as CSV
  • can assign individual point values to questions

Bad Features:

  • no simple way to insert images, video, or audio; the test creator needs to know HTML
  • the only way to identify users is through a name created by the user
  • customizability for questions is limited

Missing Features:

  • a question bank; any question the administrator makes will be on the test
  • a time limit for the test
  • ways to limit when a user can take the test
  • accommodations for coding questions
  • options for viewing a user's progress in real time
