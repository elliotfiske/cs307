2.4. Creating a Question

The scenarios in this section show an instructor creating a variety of test questions. These questions are then saved and added to the question bank.

When the instructor selects 'Add Problem' in the Question Bank menu, the system displays the dialog shown in Figure 1.

Figure 1: Default Question Creator view.


By default, the dialog will be set to the multiple choice question type.

The instructor can write the desired question in the question field. The X in the top right corner exits the dialog without saving any changes. The 'Save Question' button saves the changes to the question bank.

When the instructor selects the down arrow next to the question type display box, the menu depicted in Figure 2 is shown.

Figure 2: Drop down list of question types


From this menu, the instructor can choose what type of question he wants to create, including
  • Multiple Choice
  • True/False
  • Multiple Response
  • Short Answer
  • Essay
