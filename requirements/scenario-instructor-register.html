2.1.2. Instructor Account Registration

If the instructor selects the Instructor tab at the top of the Registration page, they are presented with the Instructor Registration page. This screen can be seen in Figure 1.

Figure 1: Instructor Registration screen.


The Registration window has two tabs at the top of it labeled "Student" and "Instructor". Selecting either of the tabs takes the user to the proper Account Registration window. The window opens with the Student tab selected by default. The Poly Username text field is for the instructor to enter their Cal Poly Portal username into for use in Test Tool. The Test Tool Password and Re-enter Password text fields are for the instructor to create a password unique to their Test Tool account and re-enter it for verification. The Instructor Code text field is where an instructor enters their Instructor Code. The Instructor Code unlocks all of the Instructor features for the instructor and is provided by a Test Tool administrator. The Submit button can be clicked after all fields are populated to create the account for the instructor.
