2. Functional Requirements

The primary functionality of Test Tool is for creating and distributing tests. Test creation is performed by administrative users, while students use the system to take the tests that are created by the instructors. Test Tool provides many functionalities that ease the test creation process.

Details of system use are presented in the following scenarios:

  • a user explores the logging-in and registering process
  • an instructor explores the home page of Test Tool
  • an instructor creates a new course
  • an instructor creates a new question
  • an instructor views student information
  • an instructor manages tests
  • a student views his current courses
These scenarios cover all functional requirements for the system.
