2.??.2. Editing a Test

In the Test Directory, the user selects a single test, then selects Properties. The left side of the window changes.



Figure ?: Properties menu.


The Title field takes a string representing the test's title.

The Course field details which course the test falls under.

The Time Limit field details the given time limit for a test. If the time remains zero, there is no time limit.

The user can save changes by clicking Save, or cancel changes by clicking Cancel.
