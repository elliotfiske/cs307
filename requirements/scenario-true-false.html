2.4.2. Creating a True/False Question

To create a true/false question, the instructor opens the question creator dialog, and then changes the question type to True/False. The instructor will see Figure 5.

Figure 5: Default True/False view.


For true/false questions, the instructor changes which is the correct option by selecting the circle next to the correct choice.
